# stdx.encoding.base64

## 功能介绍

base 包提供字符串的 Base64 编码及解码。

Base64 编码能够将二进制数据转换成仅由 64 个可打印的字符（A-Z、a-z、0-9、+、/）组成的文本格式，这使得二进制数据能够在文本环境中安全传输和存储。
